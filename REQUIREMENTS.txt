================================================================================
  INSTALLATION
================================================================================

1. Clone the repository
   ─────────────────────
   git clone https://github.com/DocFranji/shield.git
   cd shield

2. Install all dependencies (monorepo — installs everything at once)
   ──────────────────────────────────────────────────────────────────
   npm install

3. Build all packages
   ───────────────────
   npm run build

   Or build individually:
   cd packages/core    && npm run build
   cd packages/cli     && npm run build
   cd packages/mcp-server && npm run build
   cd packages/dashboard  && npm run build

================================================================================
  RUNNING SHIELD
================================================================================

Dashboard (web UI):
  cd packages/dashboard
  npm run dev
  → Opens at http://localhost:5173

CLI (scan a project):
  node packages/cli/dist/index.js scan /path/to/your/project
  node packages/cli/dist/index.js scan --quick .
  node packages/cli/dist/index.js report --format markdown

MCP Server (for Cursor / Claude Code / VS Code):
  node packages/mcp-server/dist/index.js

  Add to your IDE's MCP config:
  {
    "mcpServers": {
      "shield": {
        "command": "node",
        "args": ["/path/to/shield/packages/mcp-server/dist/index.js"]
      }
    }
  }
